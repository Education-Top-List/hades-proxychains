# Hades proxy chains
Realtime update proxylist

### Requirements
* Python 3.x
* Proxychains installed (https://github.com/haad/proxychains)
* Works on Linux


### Usage
```sh
sudo ./proxychains.sh <Command>
```

### Config proxy list
* Your list name must start with **'proxies_'** prefix.
